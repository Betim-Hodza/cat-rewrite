# CAT

this will be a simple implementation of the cat binary in C w/out the standard library.

mostly doing this for fun and will rely on the linux kernel to help me with stuff :3

plus cats!


## compilation
```bash 
gcc cat.c -Wall -fno-stack-protector -nostdlib -o cat
```
